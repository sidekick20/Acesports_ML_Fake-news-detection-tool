# Acesports_ML_Fake-news-detection-tool


A lightweight machine learning system that classifies news text as **Real** or **Fake** using NLP techniques and supervised learning.

---

## Overview

This project analyzes textual patterns (tone, structure, and word usage) to detect potential misinformation. It is designed as a quick validation layer to support user judgment—not replace it.

---

## Features

* Binary classification: Real vs Fake
* Text preprocessing with NLP
* Trained ML model for prediction
* Simple interface for input and results

---

## Tech Stack

* Python
* Scikit-learn
* Pandas, NumPy
* NLP techniques (tokenization, vectorization)

---

## Setup

```bash
git clone https://github.com/sidekick20/Acesports_ML_Fake-news-detection-tool.git
cd Acesports_ML_Fake-news-detection-tool
pip install -r requirements.txt
python app.py
```

---

## Usage

Input a news headline or article → Model predicts **Real** or **Fake** with confidence.

---

## Limitations

* Accuracy depends on dataset quality
* May misclassify satire or nuanced content
